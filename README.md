# Mobile Sales & Pricing Strategy Dashboard (Power BI)

## Project Overview
This project analyzes mobile phone sales data using **Power BI** to identify pricing strategies, revenue trends, and brand positioning insights.

The dashboard focuses on understanding how factors such as **RAM, battery capacity, customer ratings, and discounts** influence mobile phone sales performance.

---

## Tools & Technologies
- Power BI
- Microsoft Excel / CSV
- Data Visualization
- Business Analytics

---

## Business Questions
The dashboard answers several key questions:

- Which brands generate the highest revenue?
- How do RAM and battery capacity affect sales performance?
- What is the relationship between product price and customer ratings?
- How do discounts impact sales volume?
- Which brands dominate different pricing segments?

---

## Key Dashboard Insights
- Mid-range mobile phones generate the highest sales volume.
- Brands like **Apple and Samsung** lead in premium segments.
- Higher RAM and battery capacity tend to increase product pricing.
- Customer ratings have a strong impact on purchasing behavior.
- Discount strategies significantly influence product demand.

---

## Dashboard Features
The Power BI dashboard includes:

- Brand sales overview
- Customer rating impact analysis
- Mobile model comparison
- Discount vs sales performance
- Brand positioning analysis
